═══════════════════════════════════════════════════════════════════════════════
OPCIÓN 1️⃣  EJECUTAR LA APP
═══════════════════════════════════════════════════════════════════════════════

FORMA A (Más fácil):
  1. Abre Android Studio
  2. Presiona: Shift + F10

FORMA B (Línea de comandos):
  1. Abre CMD en el proyecto
  2. Ejecuta: run_on_device.bat

FORMA C (PowerShell):
  1. Abre PowerShell en el proyecto
  2. Ejecuta: Set-ExecutionPolicy -ExecutionPolicy Bypass -Scope Process
  3. Luego: .\run_on_device.ps1

¿Qué pasa? La app se compila, instala y se abre en tu emulador/dispositivo.


═══════════════════════════════════════════════════════════════════════════════
OPCIÓN 2️⃣  CONECTAR CON GITHUB
═══════════════════════════════════════════════════════════════════════════════

PASO 1: Crear repositorio (5 minutos)
  1. Ve a: https://github.com/new
  2. Nombre: TManager
  3. Click "Create repository"

PASO 2: Conectar tu proyecto (2 minutos)
  Abre terminal en tu proyecto y ejecuta:

  git remote add origin https://github.com/tu_usuario/TManager.git
  git branch -M main
  git push -u origin main

PASO 3: ¡Listo! Tu código está en GitHub

¿Alternativa? Usa GitHub Desktop (más fácil):
  • Descarga: https://desktop.github.com
  • Abre tu proyecto
  • Haz push automáticamente


═══════════════════════════════════════════════════════════════════════════════
OPCIÓN 3️⃣  DESARROLLAR Y HACER CAMBIOS
═══════════════════════════════════════════════════════════════════════════════

GUARDAR CAMBIOS LOCALES:
  git add .
  git commit -m "Descripción de lo que cambiaste"

SUBIR A GITHUB (si conectaste):
  git push origin main

VER TUS COMMITS:
  git log --oneline

VER ESTADO:
  git status
