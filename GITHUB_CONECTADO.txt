✨ INTEGRACIÓN CON ANDROID STUDIO
═════════════════════════════════════════════════════════════════════════════════

Android Studio puede hacer push/pull directamente:

1. VCS > Git > Commit
   (ve cambios, escribe mensaje, haz commit)

2. VCS > Git > Push
   (sube cambios a GitHub)

3. VCS > Git > Pull
   (trae cambios de GitHub)

Atajos:
  Commit:    Ctrl + K
  Push:      Ctrl + Shift + K
  Pull:      Ctrl + T

IMPORTANTE: Si Git te pide contraseña al hacer push/pull:

OPCIÓN 1: Personal Access Token (Recomendado)
  1. Ve a: https://github.com/settings/tokens
  2. Crea nuevo token
  3. Dale permisos: repo (full control)
  4. Copia el token
  5. Úsalo como contraseña

OPCIÓN 2: GitHub CLI
  1. Descarga: https://cli.github.com
  2. Ejecuta: gh auth login
  3. Sigue los pasos

OPCIÓN 3: SSH (Lo más seguro)
  1. Genera clave: ssh-keygen -t ed25519
  2. Agrega en GitHub: https://github.com/settings/keys
  3. Usa URL SSH: git@github.com:Baarrooso/TFG.git
